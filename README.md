# Support Ticket System

Console-based C# proof of concept for logging, viewing, closing, and prioritising support tickets.

## Features
- Create support tickets
- View open tickets
- Close tickets by ID
- Flag tickets as urgent
- Automatically set urgent tickets to High priority
- View ticket summary
- Save tickets between sessions using JSON

## Technologies
- C#
- .NET
- JSON file storage
- xUnit tests
- GitHub version control
